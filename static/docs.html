Introduction

The Trade Opportunities Analyzer API provides AI-powered market analysis for Indian sectors. Get detailed insights, opportunities, and trends for various industries using intelligent algorithms.

Base URL: https://api.tradeopportunities.com/api/v1

API Version: 1.0

Authentication

Currently, the API is public and doesn't require authentication. Future versions will support API key authentication.

Coming Soon: OAuth 2.0 and API Key authentication methods.

API Endpoints

Analyze Sector

GET /analyze/{sector}

Get detailed AI-powered market analysis for a specific sector.

Parameters

Parameter Type Required Description
sector string Yes Sector name (e.g., "technology", "agriculture")

Query Parameters

Parameter Type Default Description
detailed boolean false Include detailed market trends
ai_powered boolean true Use AI analysis (Gemini)

Supported Sectors

The API supports analysis for the following sectors:

  • Technology - Softwares, Hardware, IT Services
  • Pharmaceuticals - Drug Manufacturing, Healthcare
  • Agriculture - Farming, Agribusiness, Export
  • Banking - Financial Services, Investments
  • Automobile - Manufacturing, Auto Parts, EV
  • IT - Information Technology Services
  • Energy - Renewable, Oil & Gas, Power
  • Retail - E-commerce, Stores, Distribution
  • Infrastructure - Roads, Utilities, Construction
  • FMCG - Fast-Moving Consumer Goods
  • Telecom - Telecommunications, 5G
  • Metals - Mining, Steel, Precious Metals

Response Format

Success Response (200 OK)

{
  "sector": "technology",
  "timestamp": "2026-01-15T10:30:00Z",
  "data_sources": 15,
  "ai_powered": true,
  "report": "# Technology Sector Analysis\n\n...",
  "status": "success"
}

Response Fields

Field Type Description
sector string Analyzed sector name
timestamp datetime Analysis generation time
data_sources integer Number of data sources used
ai_powered boolean Whether AI analysis was used
report string Markdown formatted analysis report
status string Response status (success/error)

Example Requests

Basic Analysis

Request

GET /api/v1/analyze/technology

Detailed Analysis with AI

Request

GET /api/v1/analyze/agriculture?detailed=true&ai_powered=true

Using cURL

curl -X GET "https://api.tradeopportunities.com/api/v1/analyze/banking" \
  -H "Content-Type: application/json"

Using Python

import requests

response = requests.get(
    "https://api.tradeopportunities.com/api/v1/analyze/technology"
)

data = response.json()
print(data['report'])

Using JavaScript

fetch('/api/v1/analyze/retail')
  .then(response => response.json())
  .then(data => {
    console.log(data.report);
  })
  .catch(error => console.error('Error:', error));

Error Handling

Error Response Format

{
  "status": "error",
  "message": "Sector not found",
  "code": 404
}

HTTP Status Codes

Code Description Example
200 Success Analysis completed successfully
400 Bad Request Invalid sector name
404 Not Found Sector does not exist
500 Server Error Internal processing error

Common Errors

Invalid Sector: Make sure the sector name is in the supported sectors list. Check capitalization and spelling.

Rate Limiting: The API has rate limits. Please wait before making additional requests if you receive a 429 status code.
